# Agentic Research Assistant

## Overview
This project is a simple AI agent that performs multi-step research using external tools.

## Features
- Accepts research topics
- Performs step-by-step reasoning
- Retrieves information from web APIs
- Generates structured summaries

## Tech Stack
- Python
- LLM APIs
- Tavily Search API (or similar)

## Status
🚧 Basic version under development.

## Goal
To learn how AI agents perform reasoning and tool-based actions.
